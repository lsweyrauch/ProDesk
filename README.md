# ProDesk Dashboard Mock (Next.js + Tailwind)

A runnable mock of the ProDesk dashboard with:
- Light/Dark mode toggle (next-themes)
- Inline AI Assistant panel (mock API)
- KPI cards, trends chart, recent tickets, workload, system status

## Run (Replit or local)
1. Create a new **Node.js** or **Next.js** Repl.
2. Upload all files from this zip.
3. In the shell: `npm install` then `npm run dev`.
4. Open the web preview on port 3000.

_No external services required. All data is mocked._
